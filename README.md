# Camaaloth Launcher

## Description

This project is the backend of camaaloth launcher made for the Blackmagic ATEM Mini Extreme ISO.

It's using [Nest](https://github.com/nestjs/nest) framework TypeScript as framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
